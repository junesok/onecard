# 원카드 온라인

엑셀 스프레드시트로 위장한 실시간 멀티플레이어 원카드 게임.

---

## 주요 기능

- **계정 시스템** — 회원가입 / 로그인, 닉네임 및 비밀번호 변경
- **방 시스템** — 공개/비공개 방 생성 및 목록 조회, 동시에 여러 게임 진행 가능
- **실시간 멀티플레이** — 방당 최대 5명
- **순위제 진행** — 패를 다 내는 순서대로 순위가 매겨지며, 마지막 한 명이 꼴찌
- **관전 모드** — 완주한 플레이어는 남은 플레이어들의 패를 모두 볼 수 있음
- **원카드 선언** — 패가 1장 남으면 채팅창 위에 랜덤 키가 표시되며, 먼저 누른 사람이 선언 또는 신고
- **다시하기 투표** — 게임 종료 후 전원 동의 시 같은 방에서 재시작
- **주간 리더보드** — 매주 월요일 초기화, 순위·승리 수·포인트 집계
- **이탈 경고** — 게임 중 새로고침(F5) 또는 뒤로가기 시 확인 다이얼로그 표시
- **연결 끊김 처리** — 접속이 끊기면 메인 화면으로 자동 이동 및 알림
- **채팅** — 방 단위 실시간 채팅, 방 입장 시 이전 이력 자동 초기화

### 특수 카드 효과

| 카드 | 효과 |
|------|------|
| A (스페이드 제외) | 다음 플레이어 +3 드로우 |
| 스페이드 A | 다음 플레이어 +5 드로우 |
| 2 | 다음 플레이어 +2 드로우 |
| 7 | 무늬 선언 (이후 해당 무늬만 낼 수 있음) |
| J | 다음 플레이어 스킵 |
| Q | 진행 방향 반전 |
| K | 낸 플레이어가 한 번 더 (같은 무늬 또는 다른 K) |
| 흑백 조커 | 다음 플레이어 +5 드로우 |
| 컬러 조커 | 다음 플레이어 +10 드로우 |

## 설치 및 실행

```bash
git clone https://github.com/junesok/onecard.git
cd onecard
npm install
node server.js
```

서버가 시작되면 터미널에 접속 주소가 출력됩니다.
같은 네트워크의 다른 기기에서도 해당 주소로 접속할 수 있습니다.

---

## 플레이 방법

1. 브라우저에서 접속 주소 열기
2. 회원가입 또는 로그인
3. 방 목록에서 방 선택 또는 새 방 만들기
4. 2명 이상 모이면 방장이 게임 시작
5. `ESC` 키로 위장 화면 전환

---

## 데이터베이스

SQLite(`onecard.db`)를 사용하며 서버 시작 시 자동으로 생성됩니다.

| 테이블 | 설명 |
|--------|------|
| `users` | 계정 정보 (아이디, 닉네임, bcrypt 해시 비밀번호) |
| `game_results` | 게임 결과 기록 (순위, 참가 인원, 포인트, 주차) |

- WAL 모드로 동작하여 읽기/쓰기 성능 최적화
- 포인트 계산: `참가 인원 - 순위` (1등이 가장 많은 포인트)
- 리더보드는 이번 주 월요일 기준으로 집계
